# Day-1-of-100---Band-name-generator
Day 1 of 100 days of code Python challenge with Angela Yu, Udemy Course

Beginner friendly project that generates a band name based on user inputs

HOW TO RUN:
- copy the script and run in terminal, follow the prompts and enjoy your new band name

PURPOSE:

This project is part of a “100 Days of Code” challenge and focuses on:

Using print() statements
Taking user input with input()
Storing values in variables
Combining strings in output

SKILLS PRACTICED

Basic Python syntax
Variables
User interaction
String concatenation
